GIT命令行指引

您还可以按照以下说明从计算机中上传现有文件。

Git 全局设置

git config –global user.name “Cable Home”

git config –global user.email “cablefan@hotmail.com“


创建一个新仓库
git clone http://szdev.f3322.net:10095/private/mybanks.git
cd mybanks
git switch -c main
touch README.md
git add README.md
git commit -m "add README"
git push -u origin main

推送现有文件夹
cd existing_folder
git init --initial-branch=main
git remote add origin http://szdev.f3322.net:10095/private/mybanks.git
git add .
git commit -m "Initial commit"
git push -u origin main

推送现有的 Git 仓库
cd existing_repo
git remote rename origin old-origin
git remote add origin http://szdev.f3322.net:10095/private/mybanks.git
git push -u origin --all
git push -u origin --tags
--------------------------------------------------------------------------------------
  1. 切换分支：git checkout -b <branch name>
  2. 设置远程URL：git branch --set-upstream-to=origin/<branch name>
  3. 修改远程URL：git remote set-url origin http://code.79boy.com:10095/cable/guide.git
